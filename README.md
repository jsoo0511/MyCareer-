> ### Rule

1. ##### Git

   + Commit

     `[연월일 이름] 수정한 내용`

     ex ) [20200518 홍길동] Login 수정 
     
   + Branch

     + Master -> Develop -> Frontend, Backend
       + `frontend/기능`
       + `backend/기능`

2. ##### JIRA

   + Issue 생성 시, [백엔드 || 프론트엔드]를 어두에 붙이기
   
     `[프론트엔드 || 백엔드] Issue`
